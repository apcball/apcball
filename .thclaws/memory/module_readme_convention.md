# Module README Convention

- ทุก module ต้องมี `README.md` ที่ root ของ module
- สร้างใหม่ → เขียน README ครบ
- Update feature → อัปเดต README ในส่วนที่เกี่ยวข้อง + เพิ่ม changelog

## โครงสร้าง README
1. Module name + version
2. Summary
3. Features (รายละเอียด feature ทั้งหมด)
4. Dependencies
5. Technical (models, fields, files)
6. Changelog (version-based)
