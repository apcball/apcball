# IT Issue Ticket - New Required Fields

## Overview
The IT Issue ticket system has been enhanced with four new required fields to improve ticket processing and communication. These fields help IT staff better understand and resolve issues more efficiently.

## New Required Fields

### 1. Email (อีเมล)
- **Purpose**: Contact email for communication about the issue
- **Format**: Valid email address
- **Default**: Auto-populated from your employee work email
- **Required**: Yes, for Issue tickets only

### 2. ID LINE (ไลน์ไอดี)
- **Purpose**: LINE ID for quick communication via LINE messaging app
- **Format**: LINE ID (text)
- **Required**: Yes, for Issue tickets only

### 3. Symptoms/Issues (อาการเสีย)
- **Purpose**: Detailed description of the problem or symptoms
- **Format**: Text area for detailed description
- **Language**: Thai/English supported
- **Required**: Yes, for Issue tickets only
- **Tips**: 
  - Be specific about what's not working
  - Include error messages if any
  - Describe when the problem started
  - Mention any recent changes to the system

### 4. Computer Name (ชื่อคอมพิวเตอร์)
- **Purpose**: Identification of the affected computer/device
- **Format**: Computer hostname or asset tag
- **Required**: Yes, for Issue tickets only
- **How to find**:
  - Windows: Right-click "This PC" → Properties → Computer name
  - Mac: System Preferences → Sharing → Computer Name
  - Look for asset tag on physical device

## How to Fill Out an Issue Ticket

1. **Select Category**: Choose "IT Request" from the category dropdown
2. **Fill Basic Information**: 
   - Priority level
   - Your name (auto-filled)
   - Department
3. **Fill Issue Details** (NEW SECTION):
   - Email (auto-populated, can be changed)
   - ID LINE
   - Computer Name
4. **Describe Symptoms** (NEW SECTION):
   - Detailed description of the issue
   - Include error messages, screenshots if helpful
5. **Add Attachments**: Upload screenshots or relevant files
6. **Submit**: Click "Submit Issue" button

## Benefits of New Fields

- **Faster Resolution**: IT staff can contact you directly via email or LINE
- **Better Tracking**: Computer name helps identify recurring issues
- **Clear Communication**: Detailed symptoms help IT understand the problem
- **Improved Service**: All necessary information available upfront

## Example Ticket

```
Category: IT Request
Priority: Important (สำคัญ)

Issue Details:
Email: john.doe@company.com
ID LINE: johndoe123
Computer Name: PC-JOHN-001

Symptoms/Issues (อาการเสีย):
Computer is running very slow when opening Excel files.
Takes about 2-3 minutes to open a simple spreadsheet.
Error message: "Excel is not responding" appears frequently.
Started happening after Windows update last week.
```
