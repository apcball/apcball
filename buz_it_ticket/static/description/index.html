IT Ticket Management System

Overview

A comprehensive IT ticketing system for Odoo that handles three main workflows:

  • Issue/Repair: Report problems → Work in progress → Request more info (if needed) → Resolved → Closed
  • Access Request: Requester → Manager approval → IT implementation → Closed
  • Purchase Request: Requester → Manager approval → IT verification → PO creation → Receive items → Closed

Key Features

  • Separate workflows for different ticket categories
  • SLA tracking and breach notifications
  • ISO-compliant reporting
  • Dashboard with graphs and pivots
  • Multi-company support
  • Security groups with proper access rights
  • Demo data for testing

Security Groups

  • IT Requester: Employees who can create IT tickets
  • IT User: IT staff who handle tickets
  • IT Manager: IT managers with full access
  • IT Approver: Managers/HR/Finance who can approve requests
  • IT ISO Auditor: Users who can read and print ISO reports

Workflows

Issue/Repair

draft → submitted → in_progress → pending_info → resolved → closed (or cancel)

  • Submit: Create activity for IT team, set SLA
  • In Progress: IT starts work
  • Pending Info: Request more information from requester
  • Resolve: Mark as resolved, record resolution time
  • Close: Final confirmation or auto-close

Access Request

draft → submitted → waiting_manager → approved → implementing → closed (or rejected/cancel)

  • Submit: Send to manager for approval
  • Manager Approve/Reject: Manager decision
  • Start Implement: IT begins implementation
  • Close: Implementation completed

Purchase Request

draft → submitted → waiting_manager → waiting_it → po_created → received → closed (or rejected/cancel)

  • Submit: Send to manager for approval
  • Manager Approve/Reject: Manager decision
  • Create PO: IT creates purchase order from lines
  • Mark Received: Items received and delivered
  • Close: Process completed

Reports

  • Issue Report: ISO-compliant report with problem description and resolution details
  • Access Request Report: ISO-compliant report with approval signatures
  • Purchase Request Report: ISO-compliant report with PO details
  • Ticket Summary Report: Statistical report with filters for date range and categories

Dashboard

  • Ticket Counts: Open/Overdue/SLA Breach statistics by category
  • Performance Metrics: Response and resolution time averages
  • Trends: Graphs showing ticket creation over time
  • Distribution: Pivot tables showing tickets by department and category
