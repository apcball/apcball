This is a placeholder for the module icon.

To add an icon to this module:

1. Create a PNG image file named "icon.png" 
2. Recommended size: 128x128 pixels or 256x256 pixels
3. Place it in this directory: /opt/instance1/odoo17/custom-addons/marketplace_settlement/static/description/

The icon should represent marketplace settlement functionality. 
Consider using symbols like:
- Shopping cart + money/coins
- Marketplace platforms (Shopee/Lazada colors)
- Settlement/accounting symbols
- A combination of ecommerce and accounting icons

Icon design tips:
- Use clear, simple graphics
- Ensure good contrast
- Consider the Odoo app icon style
- PNG format with transparency
- Square aspect ratio
