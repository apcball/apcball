Sales Reservation Management

Enterprise-grade stock reservation system for Odoo 17

Advanced stock reservation with VIP allocation, expiration control, approval workflows, and comprehensive reporting.

Key Features

  • Customer-specific product reservations
  • VIP customer dedicated stock allocation
  • Reservation expiration with auto-release cron jobs
  • Multi-step approval workflow for stock shortage overrides
  • Real-time Available-to-Promise (ATP) calculations
  • Reservation dashboard with KPIs and charts
  • PDF reports: Reservation Detail, Aging, VIP Allocation, Customer Summary
  • Multi-company and multi-warehouse compatible
  • OWL-ready Kanban, Calendar, Pivot & Graph views
  • Granular security: User, Manager, Admin roles
  • Fully translatable with mail activity notifications

Reservation States

  • Draft - Initial state, editable
  • Waiting Approval - Requires manager approval (insufficient stock)
  • Reserved - Stock is committed
  • Allocated - Stock physically allocated
  • Delivered - Successfully delivered
  • Expired - Past expiration date
  • Released - Released back to available stock
  • Cancelled - Cancelled by user

Integration

  • Sale Order - Smart button, automatic reservation on confirmation
  • Product - On Hand / Reserved / Available display
  • Partner - VIP flag, reservation smart button
  • Stock Warehouse - Multi-warehouse support
  • Mail Activity - Notifications for expiry, shortages, approvals
