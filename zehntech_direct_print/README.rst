Direct Print Module
================================================================

Direct Print Odoo Module streamlines Odoo document printing. This module uses PrintNode API to enable instant, direct printing of all Odoo reports, invoices, and more, eliminating downloads and enhancing efficiency. Compatible with various printers, it's ideal for manufacturing, retail, sales, and any Odoo user looking to automate their printing workflow.

**Key Features**
================================================================

- **Directly Print Documents from Odoo**: Easily print invoices, delivery slips, and reports straight from Odoo to your connected printers using PrintNode. Skip manual downloads and enjoy instant, one-click printing for faster operations.
- **Secure Printer Connection via PrintNode API Key**: Connect your printers securely using the PrintNode API Key to enable direct and reliable communication between Odoo and your printing devices. Ensure fast, authenticated, and safe printing operations across your organization.
- **Default and Multi-Printer Support**: Choose a default printer for simple setups or enable multi-printer support to assign different printers for various tasks. Perfect for businesses with multiple departments or printing needs.
- **Faster and Error-Free Printing Workflow**: Streamline your document printing with one-click, direct-to-printer functionality. Eliminate manual steps and reduce printing errors for a smoother, more efficient workflow.

**Summary**
================================================================

Direct Print Odoo App allows you to instantly print Odoo reports, invoices, and documents directly to any network or local printer using PrintNode integration. Streamline your printing workflows and eliminate manual downloads.



**Installation**
================================================================

1. Download the module from the Odoo App Store or clone the repository.
2. Place the module in your Odoo addons directory.
3. Update your Odoo instance to include the new module.
4. Install the module through the Odoo interface.

**How to use this module:**
================================================================

1. Navigate to the Direct Print Module in your Odoo dashboard.
2. **Secure Printer Connection via PrintNode API Key**: Switch to PrintNode, copy your API key from the API Keys section, and paste it into the API Key field in Direct Print.
3. **Default and Multi-Printer Support**: Choose a default printer for simplified setups or enable multi-printer support to assign different printers for various tasks. Select the appropriate printer and save your settings.
4. **Directly Print Documents from Odoo**: Easily print invoices, delivery slips, and reports directly from Odoo’s modules (like Sales, Invoice, etc.).
5. **Faster and Error-Free Printing Workflow**: Streamline your document printing with one-click, direct-to-printer functionality. View your printing logs under the “Print Logs” section in PrintNode.
